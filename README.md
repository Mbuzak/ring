# ring
Render engine core

## Requirements
- CMake >= 3.12

### C libraries
- [cglm](https://github.com/recp/cglm)
- OpenGL >= 3.2
- GLEW >= 3.1
- SDL >= 2.30

## Installation (dev)
```
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Debug ..
cmake --build .
```

Download and create folder `lib/cglm`.
